# Mock Data for OKF Updater Pipeline

This directory contains sample files used to test and demonstrate the
OKF Updater Agent pipeline.

## Files

- `sales.csv` — Transaction records for product sales (10 rows)
- `employees.tsv` — Employee directory with salary info (5 rows)
- `config.json` — Application configuration in JSON format
- `config.yaml` — Application configuration in YAML format
- `schema.sql` — Database schema for customers/orders tables
- `analytics.py` — Sales analysis utility class and functions
- `inventory.db` — SQLite database with products and warehouses tables
